# 美国租金分布概况
## 概述 

本项目回答两个核心问题： 美国哪些城市的租房成本最高？哪些地区的租金涨幅最快？针对这两个问题，我基于美国社区调查（ACS）最新发布的年度数据展开分析，并完成了相关数据的地图绘制与可视化处理。我基于原作者 gboeing 提供的 notebook 与经纬度数据，下载了 2010 年和 2014 年 ACS 统计的合同租金中位数数据，同时获取了 2010 至 2014 年间各地区的居民人口年度估算数据。

上述数据集覆盖了美国所有大都市区与小都市区，这两类区域也是美国社区调查（ACS）租金数据的核心收集对象。2010 至 2014 年间，有部分大都市区的名称和识别编码发生了变更，针对这一情况，我对相关数据进行了匹配校准，确保前后数据的对应性。后续我还从数据中提取出各区域的核心城市与所属州信息，并完成了地理编码处理，将其转换为可用于地图绘制的经纬度坐标（具体处理过程可参见 Jupyter 文件）。

分析工作与静态地图的绘制，均通过 Python 的 pandas 库与 basemap 库完成；而交互式地图的制作，则采用了 Leaflet 工具实现。

## 数据来源

租金与人口数据来自美国社区调查（ACS）年度发布的微观/汇总表格（含 2010 与 2014 年度合同租金中位数与年度人口估计）。

项目起始代码与经纬度参考来自原作者（用于复现与扩展）：gboeing 提供的 notebook 与数据文件。

## 主要问题与分析目标

识别租金最高的城市，租金上涨最快的城市（排序与分位数描述）。

计算并可视化 2010 → 2014 年间的租金增长率，量化并列出“增长最快的前 10 城市”，探讨租金上涨与人口规模大是否存在关系。

将结果以地图为主的可视化形式呈现（包括静态底图与散点分布地图），便于空间模式识别与报告展示。

## 关键处理步骤

数据聚合与字段对齐 — 将 ACS 提供的不同年份表格按都会区与统一 ID合并，建立 2010/2014 的租金对照表。

名称与编码变更校准 — 针对 2010–2014 间行政区划或名称变更，采用 fuzzy join / 手工映射策略对齐记录，保证同一地理单元前后可比。

数值清洗 — 将租金字段强制为数值类型，剔除或标注异常值（如零值或极端离群点），并对缺失情况做记录与说明。

增长率计算 — 按地理单元计算百分比增长 (rent_2014 - rent_2010) / rent_2010 * 100，并对结果做分位数与截面描述。

可视化呈现 —

静态图表：用散点 / 条形 / 数据图展示分布与排名；

地理图层：在底图上用点/色块表示租金水平与增长率，并生成交互式图层供浏览器查看。

## 工具与实现要点

数据处理主要使用 pandas 与 numpy（CSV 读取、合并、分组聚合与数值计算）。

静态地图与底图绘制使用 Basemap（若改写，可使用 geopandas + contextily 以获得更现代的瓦片底图）。

交互式地图采用 Leaflet 实现（原项目通过 Leaflet 生成可交互 HTML 地图；在 Python 端可用 folium 生成等价输出）。

所有分析均在 Jupyter Notebook 中以可复现的单元格执行，并导出关键静态图表（PNG）与 HTML。

静态图表（示例：租金、人口分布散点图、Top10 增长城市柱状图、散点汇聚地图、数据表），已保存于 images；

交互式地图 HTML（outputs/result.html），便于查看租金高低分布和人口高低分布；

可复现的 Jupyter Notebook（含数据清洗脚本、分析代码与注释）。


## 结论
租金的涨跌受多重因素影响。就业岗位的增减、地方经济的繁荣与衰退，都会作用于租金水平。美联储的货币政策与通胀也会影响名义租金。城市若对开发密度进行限制，会直接约束住房供给，进而推高住房成本。从人口普查数据来看，租金中位数与都市区人口规模之间，并没有很强的相关性；2010 年以来租金涨幅与人口变化之间，也不存在明显关联。城市越大，租金未必越高；人口增长越快，租金涨幅也不一定更快。

人口普查所统计的，是受访居民当前每月实际支付的租金，它反映的是该都市区整体的住房成本水平，而非当前市场上正在招租的房源价格。统计范围包含了租金管制单元、非租金管制单元、长期租赁合同、新近租赁合同等各类住房。但需要注意的是，这一数据不能完全等同于当前市场上可租房源的真实价格。
